# ಧೈರ್ಯ — Dhairya 2.0
### A Kannada voice companion for cancer patients

> To hear, to share, and to find courage.

## Project Structure

```
dhairya-2.0/
├── stories/          # Kannada survivor stories (content)
├── webapp/           # Demo webapp for Sarvam pitch
├── voice-agent/      # Pipecat/LiveKit voice agent pipeline
├── assets/           # Audio files, images, branding
├── docs/             # Architecture docs, pitch materials
└── README.md
```

## Tech Stack
- **Sarvam AI** — STT (Saarika), TTS (Bulbul), Chat (Sarvam-M), Translation (Saaras)
- **Pipecat** — Voice agent pipeline orchestration
- **Frontend** — React webapp for demo
- **Backend** — Python/FastAPI

## Three Experiences
1. ಕಥೆ ಕೇಳಿ (Listen to Stories) — Hear survivor stories in Kannada
2. ನನ್ನ ಮಾತು ಕೇಳಿ (Hear My Story) — Speak your pain, be heard
3. ಧೈರ್ಯ ಕೊಡಿ (Give Me Courage) — Daily affirmations and strength
